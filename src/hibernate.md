<h1>Hibernate</h1>  
主流ORM 框架 Object Relation Mapping 对象映射关系, 将面向对象映射成面向关系。  
<h3>如何使用</h3>

1. 导入依赖
2. 创建 Hibernate 配置文件
3. 创建实体类
4. 创建实体类-关系映射类文件
5. 调用 Hibernate API完成操作  

<h3>注意事项</h3>  
- 创建 Maven 工厂
- 实体关系映射文件需要注册到 hibernate 配置文件之中  
- 使用Hibernate API 完成操作
- 如果你要读取 src 中的  xml文件，就需要在pom文件中配置 resources
